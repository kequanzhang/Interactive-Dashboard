#欧洲五大联赛可视化分析


项目成员：
刘计羽，张柯权，朱常怡，庄洪森

数据集：
该数据集提供了9,074场比赛，共计来自欧洲最大的5个足球联赛（英格兰，西班牙，德国，意大利，法国）从2011-2012赛季到2016-2017赛季941,009场赛事。我们用到了其中的国家，赛季，主客场，赔率，得分等等。
